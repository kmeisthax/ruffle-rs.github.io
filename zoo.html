This is 18px Palanquin Medium. It is to be used for all body copy and should be set with contrasting type color. Bold text is allowed and should be Palanquin Semibold.

Highlighted elements may be created using Brand Primary as the background color. In that case, use Hero Primary as the body copy, since it’s the highest contrast. For large runs of body copy, use Brand Secondary for better contrast against Body Primary text. These highlighted elements may otherwise be used for any design purpose.
